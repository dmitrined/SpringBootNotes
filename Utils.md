# Utils (Утилиты)

### Роль
**Utils** — это вспомогательные классы, которые содержат переиспользуемый код, не связанный напрямую с бизнес-логикой конкретной задачи.

**За что отвечает:**
- Конвертация форматов (например, даты или валюты).
- Форматирование строк.
- Математические расчеты.
- Работа с файлами (генерация путей, имен).

**Особенности:**
- Обычно это статические методы (`public static ...`).
- Класс утилит не должен хранить в себе данные.

---

### Примеры задач
1. Генерировать уникальное имя для загруженного файла.
2. Считать итоговую сумму аренды по датам.
3. Превращать `LocalDateTime` в красивую строку "15 Jan 2024".

---

### Пример кода
```java
public class FileUtils {
    // Делаем конструктор приватным, чтобы никто не создавал объект Utils
    private FileUtils() {}

    public static String generateUniqueFileName(String originalName) {
        String extension = originalName.substring(originalName.lastIndexOf("."));
        return UUID.randomUUID().toString() + extension;
    }
}
```

---

### Best Practices
1. **Private Constructor**: Обязательно добавляйте приватный конструктор, чтобы класс нельзя было инициализировать через `new`.
2. **Чистота (Pure functions)**: Старайтесь, чтобы результат метода зависел только от его входных параметров (без обращения к базе или внешним сервисам).
3. **Не раздувайте**: Если утилит становится слишком много, разделите их: `StringUtils`, `DateUtils`, `SecurityUtils`.
