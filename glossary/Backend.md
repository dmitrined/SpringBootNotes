# ⚙️ Backend & Infrastructure Terms

| Английский (Term) | Русский (Translation) | Описание (Short Explanation) |
| :--- | :--- | :--- |
| **Authentication** | Аутентификация | Проверка того, кто этот пользователь (например, по паролю). |
| **Authorization** | Авторизация | Проверка прав доступа пользователя к ресурсу. |
| **CORS** | Кросс-доменные запросы | Механизм защиты, который запрещает запросы между разными доменами. |
| **Distributed System** | Распределенная система | Система, состоящая из множества серверов, работающих вместе. |
| **Endpoint** | Эндпоинт | URL-адрес для вызова конкретной функции в API. |
| **Idempotence** | Идемпотентность | Свойство метода (напр. GET, PUT), когда повторный запрос дает тот же результат. |
| **Index** | Индекс (БД) | Структура данных для ускорения поиска в базе. |
| **Load Balancer** | Балансировщик нагрузки | Сервер, распределяющий запросы между другими серверами. |
| **Message Queue** | Очередь сообщений | Механизм передачи данных между частями системы (RabbitMQ, Kafka). |
| **Middleware** | Промежуточное ПО | Слой между запросом и ответом для логирования или обработки данных. |
| **Payload** | Полезная нагрузка | Основная часть данных в запросе или ответе. |
| **Rate Limiting** | Ограничение частоты | Лимит на количество запросов к API в минуту/час. |
| **Schema** | Схема (БД) | Структура таблиц и связей в базе данных. |
