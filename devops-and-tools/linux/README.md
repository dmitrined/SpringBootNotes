# Linux 🐧

Базовые навыки работы с Linux необходимы любому бэкенд-разработчику или DevOps-инженеру. Большинство серверов работают на Linux.

## Содержание

### Файловая система
* **Структура** — `/`, `/home`, `/etc` (конфиги), `/var/log` (логи), `/usr`, `/tmp`.
* **Абсолютные (`/home/user`) и относительные (`../`) пути.**

### Основные команды

| Команда | Описание |
| :--- | :--- |
| `ls -la` | Список файлов (включая скрытые, с размерами). |
| `cd <dir>` | Перейти в директорию. |
| `pwd` | Показать текущую директорию. |
| `mkdir -p dir/sub` | Создать директорию (включая родительские). |
| `rm -rf <dir>` | ⚠️ Удалить директорию рекурсивно (необратимо!). |
| `cp -r src dst` | Копировать файл/директорию. |
| `mv src dst` | Переместить или переименовать. |
| `cat <file>` | Вывести содержимое файла. |
| `less <file>` | Постраничный просмотр файла. |
| `grep -rn "term" .` | Найти строку в файлах рекурсивно. |
| `find . -name "*.log"` | Найти файлы по шаблону. |
| `tail -f /var/log/app.log` | Следить за логом в реальном времени. |

### Права доступа (Permissions)
* **Формат** — `-rwxr-xr--` (тип + owner + group + others).
* **`chmod 755 script.sh`** — 7=rwx, 5=r-x, Изменить права.
* **`chown user:group file`** — Изменить владельца файла.
* **`sudo <cmd>`** — Выполнить команду с правами суперпользователя.

### Процессы и управление

| Команда | Описание |
| :--- | :--- |
| `ps aux` | Список всех процессов. |
| `top` / `htop` | Интерактивный мониторинг процессов. |
| `kill -9 <PID>` | Принудительно завершить процесс по PID. |
| `lsof -i :8080` | Найти, какой процесс занимает порт 8080. |

### SSH (Secure Shell)
* **`ssh user@server-ip`** — Подключиться к удалённому серверу.
* **SSH-ключи** — Генерация: `ssh-keygen -t ed25519`. Добавить публичный ключ на сервер: `~/.ssh/authorized_keys`.
* **SCP** — Копировать файлы через SSH: `scp file.jar user@server:/app/`.

### Переменные окружения (Environment Variables)
* **`export VAR=value`** — Задать переменную в текущей сессии.
* **`.env` файлы** — В приложениях (Docker, Spring Boot).
* **`printenv`** / `echo $VAR` — Посмотреть переменные.

### Bash-скрипты
* **Shebang** — `#!/bin/bash` — первая строка скрипта.
* **Условия** — `if [ -f "file.txt" ]; then ... fi`.
* **Циклы** — `for i in *.java; do echo "$i"; done`.
* **Cron** — Планировщик задач: `crontab -e`. Формат: `* * * * * command` (минута, час, день, месяц, день недели).
