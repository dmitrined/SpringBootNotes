# 🎨 Frontend & Web Terms

| Английский (Term) | Русский (Translation) | Описание (Short Explanation) |
| :--- | :--- | :--- |
| **Asset** | Ресурс | Любой файл, используемый на странице (картинка, шрифт, скрипт). |
| **Bundle** | Бандл (пакет) | Сборка всех файлов проекта в один (или несколько) JS-файлов. |
| **Cascading** | Каскадность | Приоритетность стилей в CSS. |
| **DOM (Document Object Model)** | Объектная модель документа | Иерархическое дерево всех элементов HTML-страницы. |
| **Event Loop** | Цикл событий | Механизм, который управляет очередью задач в JavaScript. |
| **Framework** | Фреймворк | Набор инструментов и правил для разработки приложения (React, Vue, Angular). |
| **Hydration** | Гидратация | Процесс «оживления» статичного HTML на стороне клиента (в React/Next.js). |
| **Mutation** | Мутация | Изменение состояния объекта напрямую (в JS этого обычно избегают). |
| **Props (Properties)** | Пропсы (свойства) | Данные, передаваемые от родительского компонента к дочернему. |
| **Polyfill** | Полифил | Код, реализующий функциональность, которой нет в старых браузерах. |
| **Rendering** | Рендеринг | Процесс отрисовки страницы в браузере. |
| **Responsive Design** | Адаптивный дизайн | Способность страницы подстраиваться под разные экраны. |
| **Side Effect** | Побочный эффект | Изменение чего-то вне функции (например, запрос к API внутри функции). |
